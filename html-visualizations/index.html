Moderne HTML visualizations

Single-file, browser-only tools that let you drop a Moderne data-table CSV and get an interactive chart. No Python or server needed — open any HTML file in a modern browser.

All processing runs locally; CSV data never leaves your machine (except the optional EPSS lookup in the dependency-vulnerabilities tool, which queries api.first.org).

50 tools grouped by domain. Click any card to open the tool. Use each tool's Download HTML button to export a standalone single-chart HTML you can email.
