Moderne HTML visualizations

Single-file, browser-only tools that let you drop a Moderne data-table CSV and get an interactive chart. No Python or server needed — open the HTML file in any modern browser.

Each tool runs all processing locally; CSV data never leaves your machine (except the optional EPSS lookup in the vulnerability tool, which queries api.first.org).

Implemented

Click any card to open the tool. Use the in-tool download button to export a single-chart HTML you can email.
